# Meander Web — Lawn Care Template
Local business website template for lawn care businesses.
Built with vanilla HTML, CSS, and JavaScript.

## Usage
Replace all [BRACKET] tokens with client-specific content.
Deploy to Cloudflare Pages.

## Tokens to Replace
- [BUSINESS_NAME]
- [PHONE]
- [EMAIL]
- [ADDRESS], [CITY], [STATE], [ZIP]
- [TAGLINE]
- [YEAR_FOUNDED]
- [LICENSE_NUMBER]
- [PRICE_START]
- [BRAND_1], [BRAND_2]
- [BOOKING_URL] (remove button if N/A)
- [MAPS_EMBED_URL] — replace with:
  https://maps.google.com/maps?q=[ADDRESS]+[CITY]+[STATE]&output=embed
